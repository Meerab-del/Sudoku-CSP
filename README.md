# Sudoku Solver using AI 

##  Overview
This project implements an intelligent Sudoku solver using Constraint Satisfaction Problem (CSP) techniques.

##  Features
- Solves Sudoku puzzles of varying difficulty  
- Uses AC-3 (Arc Consistency) algorithm  
- Uses Backtracking with MRV heuristic  
- Applies Forward Checking for optimization  

##  AI Concepts Used
- Constraint Satisfaction Problem (CSP)  
- AC-3 Algorithm  
- Backtracking Search  
- Minimum Remaining Values (MRV)  
- Forward Checking  

##  Project Structure
- solver.py
- boards

##  How to Run
1. Place input boards in boards/ folder  
2. Run:
   python solver.py  

##  Output
- Prints original board  
- Displays solved board  
- Shows backtracking statistics
